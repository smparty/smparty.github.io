windows 常用最佳软件推荐

2019-12-31

Windows 平台同类中的优质软件，为避免选择困难，以下所有分类有且仅很少的软件，仅代表个人观点，如果你有更好的，可以在下方评论让更多人可以参考。

浏览器
360极速浏览器V12俄罗斯版
特别推荐，个人认为最好的浏览器，兼容性是最好的。同步功能也非常好用，多平台或者多台电脑使用俱佳，可以同步书签、扩展。

影音播放器
PotPlayer
特别推荐，个人认为最简单好用的本地视频播放器，民间皮肤丰富多样，总有你喜欢的风格。

下载工具
IDM
Internet Download Manager 必备最佳网络下载利器，网页嗅探高速下载无所不能，缺点是不能下载磁力和种子，可以用迅雷极速版交替使用。

迅雷X版
2018年6月推出，X版全面升级界面、框架、下载引擎、选项设置、对相关下载信息进行强化。

办公工具
Office2010
WPS Office, 永久的免费办公专家，绿色便携版是网友制作去除了联网功能，专注于办公，变得更加好用

万兴PDF专家
办公必备的免费小工具百宝箱 PDF 工具 / 万能格式转换 / 识别工具 / 图片处理 /

即时通讯
腾讯 TIM
轻聊的 QQ：风格简约，QQ 好友，消息无缝同步。 滤掉了大量 QQ 中的个性化内容和功能、界面干净简洁、操作上轻量易用、提供批量整理好友减少无效社交、邮件日程名片功能等都能看出 TIM 想营造一个纯粹的办公环境。

远程控制
AnyDesk 官方下载
AnyDesk 是一款号称速度最快的免费远程连接 / 远程桌面控制软件 实测在双方同等网络情况下，是 QQ 的 N 倍

文本编辑器
Notepad++
推荐用其替代系统自带的记事本，一款强大的轻量级开源文本编辑器，支持大部分主流语言的语法高亮显示。支持插件众多。

压缩工具
WinRAR
压缩文件管理器，知名解压缩软件，电脑装机必备软件，国内最流行最好用的压缩文件管理器、解压缩必备软件。

格式转换工具
格式工厂
格式工厂从 2007 年至今一直未停止更新，并且一直是免费软件，转换格式用格式工厂很简单便捷。一般碰到有喜欢的视频如果想转换成网页可以播放的视频格式，可以用到格式工厂，并且支持影音剪辑、压缩等多功能操作。

系统工具
火绒安全
电脑是否中毒和使用习惯有关。安全软件最大的作用并不是杀毒，而是给你一个安心。让你可以无所顾忌的去玩耍。如果非要用安全软件何不安装一个存在感不强的安全软件呢？

360驱动大师
驱动安装更新软件，百万级的驱动库，驱动安装和升级一键化，无需手动操作；首创驱动体检技术，驱动精确识别匹配；支持无人值守安装驱动，启动参数-q；首创一键智能识别假显卡、假硬件。

Dism++
最强的Windows实用工具，系统补丁更新工具，系统精简工具，系统优化工具。

阅读器
Calibre 阅读器
Calibre (电子阅读器) 是一个免费的电子图书阅读器，是一个完整的电子图书馆，包括图书馆管理，格式转换，新闻，将材料转换为电子书，以及电子书阅读器同步功能，整合进电子图书阅读器

搜索工具
Everything
Everything 是速度最快的文件名搜索工具，其速度之快令人震惊，百 G 硬盘几十万个文件，可以在几秒钟之内完成索引；文件名搜索瞬间呈现结果。它小巧免费，界面简洁，实时跟踪更新，轻松分享文件索引，支持通配符、正则表达式，可以通过 HTTP 或 FTP 分享搜索结果

屏幕录制工具
Bandicam
（班迪录屏）高清录屏软件，电脑屏幕录像软件，高清视频录像软件。Bandicam是一款简单好用的录屏大师，录屏幕，录游戏，录视频的功能强大的屏幕录像软件。

图片处理
Adobe_Photoshop_CS3
Adobe Photoshop 不需要过多介绍，绿色轻量精简版，经典实用体积小整体速度快。

FTP 工具
FlashFxp – 老牌 FTP/FXP 传输必备软件，功能强大最常用的 FTP，FXP，FTPS，SFTP 上传下载必备工具，它可以快速部署和维护用户网站。支持上传和下载站点文件，例如文档，图片，视频，音乐等等。同样也支持转移和备份本地和远程的文件，以及 FXP 服务器之间的传输和转移。支持多语言，包括简体中文在内。

拼音输入法
搜狗拼音输入法是装机必备软件，其打字准、词库全、功能强大、使得输入更高效。搜狗输入法PC版去除广告文件精简优化后的用着还不错，至少比官方版用着安心些。

微软运行库合集
微软常用运行库合集VisualCppRedist
例子比如 QQ、一些大型游戏等提示需要安装 vc 运行库，这个合集提供了所有版本的运行库，直接安装即可解决问题。
